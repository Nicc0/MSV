### Usage ###

```html
<script src="js/msv.min.js"></script>
```

```javascript
var skinViewer = new MSV("https://minotar.net/skin/");
    skinViewer.setContainer(document.body);
    skinViewer.setSkinName("Nicc0");
    skinViewer.init();
```
